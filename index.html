Konteks Penelitian

Depth First Search dalam Penyelesaian Labirin

Depth First Search adalah algoritma penelusuran graf yang mengeksplorasi setiap cabang sejauh mungkin sebelum melakukan penarikan mundur (backtracking). Saat diterapkan pada labirin, DFS menganggap setiap sel yang terbuka sebagai node dan setiap pasangan sel berdekatan yang terbuka sebagai edge tak berarah, sehingga membentuk sebuah graf implisit.

Struktur data utamanya adalah Stack LIFO (Last-In, First-Out): saat DFS maju ke sel baru, sel tersebut dimasukkan (push) ke dalam stack. Saat menemui jalan buntu (tidak ada tetangga yang belum dikunjungi), algoritma ini akan mengeluarkan (pop) isi stack dan kembali ke sel sebelumnya — inilah yang disebut sebagai tahapan backtracking.

Aplikasi ini menyediakan tampilan yang transparan dan bertahap dari proses tersebut, termasuk melihat status stack secara real-time, jumlah node yang dikunjungi, frekuensi backtrack, dan waktu eksekusi yang dihitung secara murni di luar jeda animasi.

Pseudocode DFS
procedure DFS(maze, start, finish):
  stack ← LIFO Stack baru
  visited ← Set kosong
  parent ← Map kosong

  stack.push(start)
  visited.add(start)

  while stack tidak kosong:
    current ← stack.peek()     // LIFO

    if current == finish:
      return reconstruct_path(parent)

    neighbor ← first_unvisited(current)

    if neighbor ada:
      stack.push(neighbor)
      visited.add(neighbor)
      parent[neighbor] ← current
    else:
      stack.pop()              // Backtrack
